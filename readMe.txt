# Introction

This is a demo app to show Github users and their profile.

This project is builded with MVVM architecture.

## API Document
[Github API Document](https://docs.github.com/en/rest/reference/users)

## Setting
If you cannot load data, it may be caused by http-request restrictions. Assign your `Github token` to `Constants.TOKEN` to slove it.

You can creat personal token [here](https://github.com/settings/tokens).

## 3rd party libs
- retrofit
- glide
- kotlinx-coroutines
